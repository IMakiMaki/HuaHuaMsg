
# HuaHua Msg Couple Space

This is a simple static couple space website that stores all data in the visitor's browser localStorage. It allows you and your partner to set up your names, start date, love declaration, leave messages, and record a timeline of events. The data never leaves your device.

## Features

- Set couple name, individual names, start date, and love declaration.
- Calculates and displays how many days you've been together.
- Leave messages for each other.
- Add events to a shared timeline with date, title and description.
- All data stored in browser localStorage.
- Responsive, simple design.

## Deployment

This site is ready to be deployed via GitHub Pages using the included GitHub Actions workflow. The `.nojekyll` file disables Jekyll processing.
